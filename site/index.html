Qué hace

Tu tablero de instrumentos para la IA

Cada plataforma tiene sus propias ventanas — sesiones de 5 horas, topes semanales, créditos, búsquedas restantes. TokenYou las normaliza en un solo panel legible de un vistazo.

Medidores reales, no estimaciones

Lee el mismo dato de cuota que cada plataforma le muestra a su propia interfaz: porcentaje exacto usado y hora de reinicio, por ventana y por modelo.

Cuenta regresiva de cada reset

"Reinicia en 2 h 05 m" al lado de cada barra. El badge del ícono muestra tu peor medidor con su color — verde, ámbar o rojo — sin abrir nada.

Mostrá solo lo que te importa

Ocultá los medidores que no usás con un click. Los ocultos desaparecen del panel y del badge, así el número que ves siempre es el que te afecta.

Claude Code incluido gratis

El pool de límites de Anthropic es compartido: el mismo medidor de claude.ai refleja también lo que consumís desde Claude Code y la app de escritorio.

Agregá tus propios servicios

¿Usás Cursor, Copilot u otra herramienta con cuota? Definila como JSON en 30 segundos, sin tocar el código de la extensión, y aparece como una plataforma más.

Seguridad

Diseñada para que no tengas que confiar en nosotros

✓

Incapacidad arquitectónica

TokenYou no "promete" no leer tus chats: no puede. Ningún script corre dentro de ninguna página. Solo hace fetches en segundo plano al endpoint de cuota de cada plataforma y lee números.

✓

Tu sesión nunca se toca

Las peticiones son same-origin: el navegador adjunta la sesión por su cuenta. La extensión no tiene permiso cookies y no puede ver la credencial.

✓

Permiso por plataforma, revocable

Recién instalada, TokenYou no accede a ningún sitio. Activás Claude, ChatGPT o la que uses, una por una, y podés quitar cualquiera cuando quieras.

✓

Verificable, no solo auditable

El paquete se genera con git archive desde un commit etiquetado: mismo commit, mismo zip, hash publicado. Cualquiera comprueba que lo instalado es exactamente este código.

Cómo leer los medidores

Verde, ámbar, rojo — y dos marcas en cada barra

Todos los medidores usan la misma escala semántica, y cada barra lleva dos marcas de umbral fijas (en 60% y 85%) para que veas cuánto te falta para la próxima zona. El badge del ícono toma el color de tu peor medidor visible.

0 – 59% · verde

Zona cómoda. Usá con libertad: tenés margen de sobra en esta ventana.

60 – 84% · ámbar

Zona de atención. Conviene dosificar o mirar cuánto falta para el reset antes de una tarea larga.

85 – 100% · rojo

Zona crítica. Estás por chocar con el límite: guardá lo importante o cambiá de plataforma.

FAQ

Preguntas frecuentes

¿TokenYou puede leer mis conversaciones?

No, y no por una promesa sino por arquitectura: ningún código de TokenYou corre dentro de ninguna página. Solo hace peticiones en segundo plano al endpoint de cuota de cada plataforma — el mismo que esa plataforma usa para mostrarte tu propio uso — y lee números: porcentajes, restantes y horas de reset.

¿Por qué me pide permiso para cada sitio por separado?

Principio de mínimo privilegio: recién instalada, TokenYou no tiene acceso a ningún sitio. Cada plataforma que activás es un permiso individual que Chrome te muestra y que podés revocar en cualquier momento con el botón "Quitar". Si nunca activás una plataforma, la extensión jamás podrá tocarla.

¿Manda algún dato a algún servidor?

No existe ningún canal de salida: cero telemetría, cero analytics, cero servidores propios, cero dependencias externas. El único tráfico de red es hacia las plataformas que vos activaste, y únicamente a sus endpoints de cuota. Todo se guarda en storage.local de tu navegador.

¿Cómo verifico que lo publicado es este código?

El paquete se genera con git archive desde un commit etiquetado, lo que lo hace byte-reproducible: cloná el repo, ejecutá el mismo comando y compará el hash SHA-256 con el publicado en cada release. El CI de GitHub lo genera automáticamente.

¿Qué pasa cuando una plataforma cambia su API interna?

Los endpoints de cuota no son contratos públicos y pueden cambiar. Cada plataforma vive en un adaptador aislado: si una rompe, esa card muestra "no disponible" y el resto sigue funcionando. Al ser open source, el arreglo suele llegar en días — y podés reportarlo o contribuirlo vos en GitHub.

¿Sirve para trackear Claude Code, Cursor o Copilot?

Claude Code sí, automáticamente: comparte el pool de límites con claude.ai, así que el medidor de Claude ya lo incluye. Cursor y GitHub Copilot se agregan como servicios personalizados desde la página de opciones (hay plantillas listas), leyendo sus dashboards web con tu sesión.

¿Es gratis? ¿Cuál es el modelo de negocio?

Gratis y MIT. No hay modelo de negocio oculto: no se venden datos (no hay datos), no hay premium, no hay cuentas. Es software de la comunidad, parte del Ecosistema Escriba.

¿Ayuda a saltarse los límites de las plataformas?

No. TokenYou es un monitor de solo lectura de tu propia cuota — te muestra lo que las plataformas ya te muestran, mejor organizado. No elude, extiende ni manipula ningún límite, en línea con las políticas del Chrome Web Store.

Acerca de

Parte del Ecosistema Escriba

TokenYou es parte del Ecosistema Escriba, la familia de herramientas abiertas construidas sobre una misma convicción: el software que convive con tus datos más sensibles debe ser verificable, no solo confiable. Cada pieza del ecosistema es open source, procesa localmente todo lo que puede y trata los permisos como algo que se gana, no que se asume.

TokenYou nació de una frustración simple — chocar con el límite de Claude a mitad de una idea — y de una observación incómoda: las extensiones que resolvían esto pedían más acceso del que necesitaban, con código cerrado, junto a las conversaciones más valiosas que tenemos con una IA. La respuesta fue construir el medidor que queríamos usar: uno que no puede espiar ni aunque quisiera.
